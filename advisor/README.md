## Use Cases

### 1. Before Sending Important Emails

**Problem:** About to send email asking "Is it okay if I..."

**Advisor helps:**
- Detects permission-seeking language
- Reframes to outcome-driven communication
- Helps craft strategic questions for real help

**Example:**

### 2. Setting Goals/OKRs

**Problem:** Goals are all tasks: "Set up dashboard", "Write docs"

**Advisor helps:**
- Identifies low-altitude (task vs outcome)
- Challenges to think 10x bigger
- Focuses on business impact

**Example:**

### 3. Crafting Questions to Manager

**Problem:** Not sure what to ask, or asking wrong questions

**Advisor helps:**
- Tests if question is strategic vs permission-seeking
- Provides question templates
- Ensures you're asking for help, not approval

**Example:**
```

organizational dynamics am I missing that would change this?"
```

### 4. When Feeling Stuck/Uncertain

**Problem:** Waiting for direction, not sure what to do next

**Advisor helps:**
- Identifies "waiting mode" mentality
- Pushes to identify opportunities proactively
- Forces ownership of next steps

**Example:**
```
From: "What should I focus on next?"
To: "I've identified [opportunity] based on [signals]. I'm driving
toward [outcome]. Starting with [action] today."
```
